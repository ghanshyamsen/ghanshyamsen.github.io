PHP 8.3 Features Every Developer Should Know

Modern PHP features that make your code cleaner, safer, and more performant.

PHP continues to evolve with each release, and PHP 8.3 brings exciting new features that make our code cleaner, safer, and more performant. Let's explore the most impactful additions you should start using today.

1. Typed Class Constants

Finally! Class constants can now have explicit type declarations.

class PaymentGateway
{
    public const string CURRENCY = 'USD';
    public const int MAX_RETRY = 3;
    public const array SUPPORTED_METHODS = ['card', 'bank', 'wallet'];
}

class StripeGateway extends PaymentGateway
{
    // This would cause an error - type mismatch!
    // public const string MAX_RETRY = 'three'; 
    
    public const int MAX_RETRY = 5; // Valid override
}

2. json_validate() Function

No more decoding just to check if JSON is valid:

// Old way - wasteful
function isValidJson(string $json): bool
{
    json_decode($json);
    return json_last_error() === JSON_ERROR_NONE;
}

// New way - PHP 8.3
$json = '{"name": "Ghanshyam", "role": "developer"}';

if (json_validate($json)) {
    $data = json_decode($json, true);
}

3. Dynamic Class Constant Fetch

class Status
{
    public const PENDING = 'pending';
    public const APPROVED = 'approved';
    public const REJECTED = 'rejected';
}

$statusName = 'APPROVED';
$status = Status::{$statusName};  // Returns 'approved'

4. #[\Override] Attribute

Mark methods as overriding parent methods. The compiler will error if the parent method doesn't exist:

class BaseController
{
    public function authorize(): bool
    {
        return true;
    }
}

class AdminController extends BaseController
{
    #[\Override]
    public function authorize(): bool
    {
        return $this->user->isAdmin();
    }
}

5. Randomizer Additions

$randomizer = new Random\Randomizer();

// Get random bytes as hex string
$token = $randomizer->getBytesFromString(
    '0123456789abcdef',
    32
);

// Random float between 0 and 1
$percentage = $randomizer->getFloat(0, 1);

Migration Tips

  1. Run your test suite with deprecation warnings enabled
  2. Add type declarations to class constants gradually
  3. Replace json_decode validation patterns with json_validate()
  4. Consider adding #[\Override] to catch typos early
